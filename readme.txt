Shorten your wordpress URLs using Goo.gl URL shortener, also support Google Authentication for tracking analytics.

== Description ==
WP Goo.gl URL shortener plugin for wordpress allows you to shorten your wordpress URLs using Goo.gl.

This plugin will work with and with out Google authentication.

This plugin supports authentication to shorten your urls using your Google Account, so you can track the nice analytics of your URLs.

== Installation ==
1. Upload the plugin to your plugins directory.
2. Goto the plugin menu and activate it.
3. (Optional) If you need to track analytics, goto WP Goo.gl settings page and Enable Google Authentication
